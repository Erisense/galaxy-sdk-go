结构化存储Go SDK
========================
1. 配置go环境变量GOPATH，并将sdk代码解压到对应目录(或直接通过go get安装)
2. 修改basic.go中的app key/secret

```
cd examples
go run basic.go
```

SDS Go SDK User Guide
========================
1. Set up GOPATH, install SDK manually or by `go get`
2. change the app id/secret in the example code

```
cd examples
go run basic.go
```
